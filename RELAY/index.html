ℹ️ About dMarchFF Relay Pro

dMarchFF Relay Pro is a professional-grade IoT control panel designed for seamless management of relay-based devices. Built for reliability, scalability, and ease of use — empowering smart homes, industrial automation, and commercial deployments.

✨ Key Features
  • Real-time MQTT-based relay control (up to 16 channels)
  • Cloud-synced scheduling with GitHub integration
  • Live system monitoring: RSSI, uptime, temperature, firmware
  • OTA firmware updates & remote device management
  • Responsive web UI — works on desktop, tablet & mobile
  • Secure WebSocket (WSS) support with authentication
